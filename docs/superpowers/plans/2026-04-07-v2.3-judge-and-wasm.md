# v2.3 Haiku LLM Judge + WASM Sandbox Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add Tier 3 (Haiku LLM judge for ambiguous commands) and Tier 4 (WASM sandbox via Wasmtime for isolated code verification), completing the 5-tier cascade verification architecture from ADR-004.

**Architecture:** The Haiku judge is called ONLY when the semantic router returns `RoutingHint::LlmJudge` (ambiguous commands). It calls Claude Haiku via the Anthropic API with a structured safety prompt, caches results in an LRU, and respects a per-session budget (default 20 calls). The WASM sandbox provides a `WasmSandbox` alongside the existing `ProcessSandbox`, using Wasmtime for capability-based isolation of Python/JS verification code. Both features are opt-in and feature-gated.

**Tech Stack:** reqwest (already a dep) for Anthropic API calls, lru crate for cache, wasmtime v43 for WASM runtime

```rust
//! Tier 3: Haiku LLM Judge for ambiguous commands.
//!
//! Called ONLY when the semantic router returns `RoutingHint::LlmJudge`.
//! Opt-in: requires `ANTHROPIC_API_KEY` environment variable.
//! Budgeted: max N calls per session (default 20).
//! Cached: LRU cache avoids repeated API calls for identical commands.

use std::sync::Mutex;
use std::time::Duration;

use lru::LruCache;
use std::num::NonZeroUsize;

/// Configuration for the Haiku judge.
#[derive(Debug, Clone)]
pub struct JudgeConfig {
    /// Max API calls per session (default: 20).
    pub max_calls_per_session: u32,
    /// Timeout for API call in ms (default: 500).
    pub timeout_ms: u64,
    /// LRU cache size (default: 100).
    pub cache_size: usize,
    /// Model to use (default: claude-haiku-4-5-20251001).
    pub model: String,
}

impl Default for JudgeConfig {
    fn default() -> Self {
        Self {
            max_calls_per_session: 20,
            timeout_ms: 500,
            cache_size: 100,
            model: "claude-haiku-4-5-20251001".into(),
        }
    }
}

/// Verdict from the Haiku judge.
#[derive(Debug, Clone, Copy, PartialEq, Eq, Hash)]
pub enum JudgeVerdict {
    /// Command is safe to execute.
    Safe,
    /// Command is risky — ask for confirmation.
    Risky,
    /// Command is dangerous — block.
    Dangerous,
    /// Judge could not evaluate (no API key, timeout, budget exhausted).
    Abstain,
}

/// Haiku LLM judge for ambiguous commands.
pub struct HaikuJudge {
    config: JudgeConfig,
    api_key: Option<String>,
    cache: Mutex<LruCache<String, JudgeVerdict>>,
    calls_made: Mutex<u32>,
}

impl HaikuJudge {
    /// Create a new judge. Reads `ANTHROPIC_API_KEY` from environment.
    pub fn new(config: JudgeConfig) -> Self {
        let api_key = std::env::var("ANTHROPIC_API_KEY").ok();
        let cache_size = NonZeroUsize::new(config.cache_size).unwrap_or(NonZeroUsize::new(1).unwrap());
        Self {
            config,
            api_key,
            cache: Mutex::new(LruCache::new(cache_size)),
            calls_made: Mutex::new(0),
        }
    }

    /// Check if the judge is available (API key present).
    pub fn is_available(&self) -> bool {
        self.api_key.is_some()
    }

    /// Remaining budget for this session.
    pub fn remaining_budget(&self) -> u32 {
        let made = *self.calls_made.lock().expect("calls_made lock");
        self.config.max_calls_per_session.saturating_sub(made)
    }

    /// Manually insert a verdict into the cache (for testing).
    pub fn cache_verdict(&self, command: &str, context: Option<&str>, verdict: JudgeVerdict) {
        let key = cache_key(command, context);
        if let Ok(mut cache) = self.cache.lock() {
            cache.put(key, verdict);
        }
    }

    /// Evaluate a command. Returns a verdict.
    ///
    /// Checks cache first, then budget, then API key.
    /// Falls back to `Abstain` if any check fails.
    pub async fn evaluate(&self, command: &str, context: Option<&str>) -> JudgeVerdict {
        let key = cache_key(command, context);

        // Check cache
        if let Ok(mut cache) = self.cache.lock() {
            if let Some(&verdict) = cache.get(&key) {
                tracing::debug!(command, "judge: cache hit");
                return verdict;
            }
        }

        // Check API key
        let Some(ref api_key) = self.api_key else {
            tracing::debug!("judge: no ANTHROPIC_API_KEY, abstaining");
            return JudgeVerdict::Abstain;
        };

        // Check budget
        {
            let mut made = self.calls_made.lock().expect("calls_made lock");
            if *made >= self.config.max_calls_per_session {
                tracing::info!("judge: budget exhausted ({} calls made)", *made);
                return JudgeVerdict::Abstain;
            }
            *made += 1;
        }

        // Call API
        let verdict = self.call_api(api_key, command, context).await;

        // Cache result
        if let Ok(mut cache) = self.cache.lock() {
            cache.put(key, verdict);
        }

        verdict
    }

    /// Call the Anthropic API.
    async fn call_api(&self, api_key: &str, command: &str, context: Option<&str>) -> JudgeVerdict {
        let prompt = build_judge_prompt(command, context);
        let timeout = Duration::from_millis(self.config.timeout_ms);

        let client = match reqwest::Client::builder().timeout(timeout).build() {
            Ok(c) => c,
            Err(e) => {
                tracing::warn!("judge: failed to build HTTP client: {e}");
                return JudgeVerdict::Abstain;
            }
        };

        let body = serde_json::json!({
            "model": self.config.model,
            "max_tokens": 50,
            "messages": [
                {
                    "role": "user",
                    "content": prompt,
                }
            ]
        });

        let result = client
            .post("https://api.anthropic.com/v1/messages")
            .header("x-api-key", api_key)
            .header("anthropic-version", "2023-06-01")
            .header("content-type", "application/json")
            .json(&body)
            .send()
            .await;

        match result {
            Ok(resp) if resp.status().is_success() => {
                if let Ok(json) = resp.json::<serde_json::Value>().await {
                    parse_verdict(&json)
                } else {
                    JudgeVerdict::Abstain
                }
            }
            Ok(resp) => {
                tracing::warn!("judge: API returned HTTP {}", resp.status());
                JudgeVerdict::Abstain
            }
            Err(e) => {
                if e.is_timeout() {
                    tracing::info!("judge: API timeout ({}ms)", self.config.timeout_ms);
                } else {
                    tracing::warn!("judge: API error: {e}");
                }
                JudgeVerdict::Abstain
            }
        }
    }
}

/// Build the judge prompt.
pub fn build_judge_prompt(command: &str, context: Option<&str>) -> String {
    let ctx_line = context
        .map(|c| format!("\nTask context: {c}"))
        .unwrap_or_default();

    format!(
        "You are a security judge for an AI coding agent. \
         Evaluate whether the following shell command is safe to execute.{ctx_line}\n\n\
         Command: `{command}`\n\n\
         Answer with exactly one word: SAFE, RISKY, or DANGEROUS.\n\
         Then one sentence explaining why."
    )
}

/// Parse the API response into a verdict.
fn parse_verdict(json: &serde_json::Value) -> JudgeVerdict {
    let text = json
        .get("content")
        .and_then(|c| c.as_array())
        .and_then(|arr| arr.first())
        .and_then(|block| block.get("text"))
        .and_then(|t| t.as_str())
        .unwrap_or("");

    let first_word = text.split_whitespace().next().unwrap_or("").to_uppercase();
    match first_word.as_str() {
        "SAFE" => JudgeVerdict::Safe,
        "RISKY" => JudgeVerdict::Risky,
        "DANGEROUS" => JudgeVerdict::Dangerous,
        _ => {
            // Try to find the word anywhere in the response
            let upper = text.to_uppercase();
            if upper.contains("DANGEROUS") {
                JudgeVerdict::Dangerous
            } else if upper.contains("RISKY") {
                JudgeVerdict::Risky
            } else if upper.contains("SAFE") {
                JudgeVerdict::Safe
            } else {
                tracing::warn!("judge: could not parse verdict from: {text}");
                JudgeVerdict::Abstain
            }
        }
    }
}

/// Build a cache key from command + context.
fn cache_key(command: &str, context: Option<&str>) -> String {
    match context {
        Some(ctx) => format!("{command}|{ctx}"),
        None => command.to_string(),
    }
}
```

```rust
// Tier 3: LLM Judge — call Haiku when routing hint says LlmJudge
#[cfg(feature = "judge")]
if matches!(ctx.routing_hint, Some(metaygn_shared::state::RoutingHint::LlmJudge)) {
    // Check session budget
    let has_budget = if let Ok(sess) = session_ctx.lock() {
        sess.judge_calls_remaining > 0
    } else {
        false
    };

    if has_budget {
        let task_ctx = if let Ok(sess) = session_ctx.lock() {
            sess.task_type.map(|t| format!("{:?}", t))
        } else {
            None
        };

        let verdict = state.judge.evaluate(&cmd, task_ctx.as_deref()).await;

        // Decrement session budget
        if let Ok(mut sess) = session_ctx.lock() {
            if sess.judge_calls_remaining > 0 {
                sess.judge_calls_remaining -= 1;
            }
        }

        match verdict {
            crate::judge::JudgeVerdict::Dangerous => {
                // Override: block the command
                let mut output = HookOutput::permission(
                    PermissionDecision::Deny,
                    format!("[judge:dangerous] Haiku judge classified command as dangerous"),
                );
                append_session_budget(&mut output, &session_ctx);
                append_latency(&mut output, start);
                let resp_json = serde_json::to_string(&output).unwrap_or_default();
                record_replay(&state, &session_id, "PreToolUse", &payload, &resp_json, start).await;
                return Json(output);
            }
            crate::judge::JudgeVerdict::Risky => {
                // Override: ask for confirmation
                let mut output = HookOutput::permission(
                    PermissionDecision::Ask,
                    format!("[judge:risky] Haiku judge flagged command as risky"),
                );
                append_session_budget(&mut output, &session_ctx);
                append_latency(&mut output, start);
                let resp_json = serde_json::to_string(&output).unwrap_or_default();
                record_replay(&state, &session_id, "PreToolUse", &payload, &resp_json, start).await;
                return Json(output);
            }
            _ => {
                // Safe or Abstain — continue normally
                tracing::debug!(verdict = ?verdict, "judge: command cleared");
            }
        }
    }
}
```

```rust
//! WASM-based sandbox using Wasmtime for capability-based isolation.
//!
//! Provides an alternative to ProcessSandbox with:
//! - Faster startup (1-5ms vs 5-20ms)
//! - Cross-platform identical behavior
//! - Deny-by-default security (no FS, no network unless explicitly granted)
//!
//! Limitation: can only run WASM modules, not arbitrary shell commands.
//! For shell commands, continue using ProcessSandbox.

use std::time::{Duration, Instant};

use anyhow::{Context, Result};
use wasmtime::*;
use wasmtime_wasi::preview1::WasiP1Ctx;
use wasmtime_wasi::{WasiCtxBuilder};

use crate::SandboxResult;

/// Configuration for the WASM sandbox.
#[derive(Debug, Clone)]
pub struct WasmSandboxConfig {
    /// Max execution time in ms (default: 5000).
    pub timeout_ms: u64,
    /// Max memory in bytes (default: 64 MB).
    pub max_memory_bytes: usize,
    /// Whether to allow network access (default: false).
    pub allow_network: bool,
    /// Directories to pre-open for read (default: none).
    pub readable_dirs: Vec<String>,
    /// Directories to pre-open for write (default: none).
    pub writable_dirs: Vec<String>,
}

impl Default for WasmSandboxConfig {
    fn default() -> Self {
        Self {
            timeout_ms: 5000,
            max_memory_bytes: 64 * 1024 * 1024,
            allow_network: false,
            readable_dirs: Vec::new(),
            writable_dirs: Vec::new(),
        }
    }
}

/// WASM sandbox using Wasmtime.
pub struct WasmSandbox {
    engine: Engine,
    config: WasmSandboxConfig,
}

impl WasmSandbox {
    /// Create a new WASM sandbox.
    pub fn new(config: WasmSandboxConfig) -> Result<Self> {
        let mut engine_config = Config::new();
        engine_config.consume_fuel(true);
        // Set fuel limit proportional to timeout
        // ~1M fuel units per second is a rough estimate
        let fuel = (config.timeout_ms as u64) * 1_000_000 / 1000;

        let engine = Engine::new(&engine_config)
            .context("failed to create Wasmtime engine")?;

        Ok(Self { engine, config })
    }

    /// Execute a WAT (WebAssembly Text) program.
    pub async fn execute_wat(&self, wat: &str) -> Result<SandboxResult> {
        let wasm = wat::parse_str(wat)
            .context("failed to parse WAT")?;
        self.execute_wasm(&wasm).await
    }

    /// Execute a compiled WASM binary.
    pub async fn execute_wasm(&self, wasm: &[u8]) -> Result<SandboxResult> {
        let engine = self.engine.clone();
        let timeout_ms = self.config.timeout_ms;
        let wasm = wasm.to_vec();

        // Run in a blocking task since Wasmtime is sync
        let result = tokio::task::spawn_blocking(move || {
            let start = Instant::now();

            let module = match Module::new(&engine, &wasm) {
                Ok(m) => m,
                Err(e) => {
                    return Ok(SandboxResult {
                        success: false,
                        exit_code: None,
                        stdout: String::new(),
                        stderr: format!("WASM compilation error: {e}"),
                        duration_ms: start.elapsed().as_millis() as u64,
                        timed_out: false,
                    });
                }
            };

            // Build WASI context (deny-by-default)
            let wasi_ctx = WasiCtxBuilder::new()
                .inherit_stdout()
                .inherit_stderr()
                .build_p1();

            let mut store = Store::new(&engine, wasi_ctx);

            // Set fuel limit for timeout enforcement
            let fuel = (timeout_ms as u64) * 1_000_000 / 1000;
            store.set_fuel(fuel).ok();

            let mut linker = Linker::new(&engine);
            wasmtime_wasi::preview1::add_to_linker_sync(&mut linker, |ctx: &mut WasiP1Ctx| ctx)
                .ok();

            let instance = match linker.instantiate(&mut store, &module) {
                Ok(i) => i,
                Err(e) => {
                    return Ok(SandboxResult {
                        success: false,
                        exit_code: None,
                        stdout: String::new(),
                        stderr: format!("WASM instantiation error: {e}"),
                        duration_ms: start.elapsed().as_millis() as u64,
                        timed_out: false,
                    });
                }
            };

            // Try to call _start (WASI entry point)
            let func = instance.get_typed_func::<(), ()>(&mut store, "_start");
            let (success, stderr) = match func {
                Ok(f) => match f.call(&mut store, ()) {
                    Ok(()) => (true, String::new()),
                    Err(e) => {
                        let timed_out = e.to_string().contains("fuel");
                        if timed_out {
                            return Ok(SandboxResult {
                                success: false,
                                exit_code: None,
                                stdout: String::new(),
                                stderr: "WASM execution timed out (fuel exhausted)".into(),
                                duration_ms: start.elapsed().as_millis() as u64,
                                timed_out: true,
                            });
                        }
                        (false, format!("WASM execution error: {e}"))
                    }
                },
                Err(_) => {
                    // No _start function — module loaded but nothing to run
                    (true, String::new())
                }
            };

            let duration_ms = start.elapsed().as_millis() as u64;

            Ok::<_, anyhow::Error>(SandboxResult {
                success,
                exit_code: if success { Some(0) } else { Some(1) },
                stdout: String::new(), // TODO: capture stdout via pipe
                stderr,
                duration_ms,
                timed_out: false,
            })
        })
        .await
        .context("WASM task panicked")??;

        Ok(result)
    }
}
```

```rust
/// POST /sandbox/wasm — execute a WASM module in the sandbox.
#[cfg(feature = "wasm")]
async fn exec_wasm(
    State(_state): State<AppState>,
    Json(req): Json<WasmExecRequest>,
) -> Json<metaygn_sandbox::SandboxResult> {
    use metaygn_sandbox::wasm_sandbox::{WasmSandbox, WasmSandboxConfig};

    let config = WasmSandboxConfig {
        timeout_ms: req.timeout_ms.unwrap_or(5000),
        ..Default::default()
    };

    let sandbox = match WasmSandbox::new(config) {
        Ok(s) => s,
        Err(e) => {
            return Json(metaygn_sandbox::SandboxResult {
                success: false,
                exit_code: None,
                stdout: String::new(),
                stderr: format!("Failed to create WASM sandbox: {e}"),
                duration_ms: 0,
                timed_out: false,
            });
        }
    };

    match sandbox.execute_wat(&req.wat).await {
        Ok(result) => Json(result),
        Err(e) => Json(metaygn_sandbox::SandboxResult {
            success: false,
            exit_code: None,
            stdout: String::new(),
            stderr: format!("WASM execution failed: {e}"),
            duration_ms: 0,
            timed_out: false,
        }),
    }
}

#[cfg(feature = "wasm")]
#[derive(serde::Deserialize)]
pub struct WasmExecRequest {
    pub wat: String,
    pub timeout_ms: Option<u64>,
}
```
